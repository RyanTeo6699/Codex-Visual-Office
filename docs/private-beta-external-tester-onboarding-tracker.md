- Invite status: `not_invited`, `invited`, `declined`, `accepted`, `no_response`.

- Setup status: `not_started`, `attempted`, `passed`, `failed`, `blocked`.
- Launch status: `not_started`, `launched`, `failed`, `blocked`.
- Approved path status: `not_configured`, `configured`, `approved`, `blocked`.

- Feedback status: `not_requested`, `pending`, `submitted`, `triaged`, `closed`.
