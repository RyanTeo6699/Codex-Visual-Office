# Private Beta External Tester Consent / Safety Acknowledgment

## Private Beta Status

Codex Visual Office is currently in private beta intake. It is not a public release, not a commercial launch, not a signed installer, not notarized, and not auto-updating.

## Local-First Status

The product is local-first. Testers should run it locally from the provided setup flow. Phase 29 does not add cloud sync, team workspace, auth, payment, MCP, ChatGPT App, GitHub API, Vercel, Supabase, or OpenAI API capabilities.

## Safety Rules

The tester agrees:

- No token sharing.
- No `~/.codex/auth.json` sharing.
- No `.env` or `.env.local` sharing.
- No private key sharing.
- No password, API key, or credential sharing.
- No proprietary source package sharing.
- No local SQLite DB sharing.
- No unredacted logs with secrets.
- Screenshots and logs must be redacted before submission.

## Acceptable Feedback Evidence

- Redacted screenshot.
- Bounded and redacted log note.
- Environment summary.
- Reproduction steps.
- Route or page name.
- Observed error message without secrets.
